## 🖥️ Description

This Python script automatically simulates keystrokes in Roblox to prevent the game from kicking you for inactivity (AFK).  
It supports random and fixed intervals, and allows you to choose which keys to press.

---

## 🚀 Features

- ✅ Automatically detects and activates the Roblox window  
- ✅ Simulates `W`, `A`, `S`, `D`, and `Space` keys  
- ✅ Choose between random or fixed time intervals
