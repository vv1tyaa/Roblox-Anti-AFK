# 🎮 Roblox Anti-AFK

**Version: 1.0 - Release**

---

## 🚀 Features

- ✅ Automatically detects and activates the Roblox window  
- ✅ Simulates `W`, `A`, `S`, `D`, and `Space` keys  
- ✅ Choose between random or fixed time intervals  
- ✅ Option to set custom key hold duration

---

## ⚙️ Installation

```bash
pip install keyboard pygetwindow
```

---

## 📥 Download

[⬇️ Download Anti-AFK.py](https://raw.githubusercontent.com/vv1tyaa/Roblox-Anti-AFK/main/Anti-AFK.py)

---

## 🚀 Usage

1. Download the script using the link above.
2. Run it:
   ```bash
   python Anti-AFK.py
   ```
3. Follow the on-screen instructions to select interval and key mode.
